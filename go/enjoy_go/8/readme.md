## UNIT 38. 인터넷 저장소 패키지 사용하기

* 기존 npm, pip보다 좀 더 유연한 방법을 제공하고 있는듯. 기존 npm, pip의 경우 패키지 저장소의 있는 것들만 사용이 가능하고 나머지는 직접 소스를 내려받아 즉정 경로에 넣고 사용해야하는데 go는 기본적으로 이러한 기능을 제공하고 있음.
