# 압축 하고 풀기

## tar로 압축하기

```
$ tar -cvf <file name.tar> <folder name>
```

## tar 압축 풀기

```
$ tar -xvf <file name.tar>
``` 

## tar.gz 압축하기

```
$ tar -zcvf <file name.tar.gz> <folder name>
```
## tar.gz 압축 풀기

```
$ tar -zxvf <file name.tar.gz> <folder name>
```
