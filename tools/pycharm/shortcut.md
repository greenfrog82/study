# pycharm의 유용한 단축키 정리

### Create Empty Files 

프로젝트에 새로운 파일을 추가하고자 할 때 사용하는 단축키.
**alt + insert**

### Select Target

PyCharm에서 현재 선택 된 파일의 표시하는 화면으로 이동하는 단축키.
**Ctrl + Shift + 1** 

### Show/Hide Project View

**Ctrl + 1**
